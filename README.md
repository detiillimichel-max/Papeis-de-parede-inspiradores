📱 Prévia
Todos os wallpapers foram criados em alta resolução, prontos para celular. Tipografia elegante e fundos minimalistas.

Para ver e baixar, abra o arquivo index.html ou acesse o link do GitHub Pages acima.

🚀 Como usar
Opção 1: Pelo celular (mais fácil)

Acesse o link do site
Toque em Baixar no wallpaper que gostou
Salve na galeria e defina como papel de parede
Opção 2: Pelo repositório

Baixe o arquivo index.html
Abra no navegador
Clique com o botão direito na imagem > Salvar como
